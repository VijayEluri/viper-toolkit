Provides the classes necessary to read comparable Descriptors in from a file.

A {@link viper.descriptors.Descriptor Descriptor} is a description of some action, event, object, state, etc. found in a video file. It can be as simple as a boolean value and as complex as tracking moving text in a bounding box as it changes color. This is acomplished by specifiying {@link descriptors.attributes Attributes}. These objects can then be compared to determine accuracy, or distance.

Distance is defined as a measure of the difference between two things, with certain desireable properties. Theseproperties are:

  • Positiveness: If A != B, then dist(A,B)) > 0. If A == B, then dist(A,B) = 0.
  • Symmetry: dist(A,B) = dist(B,A)
  • Triangle Inequality: dist(A,C) < dist(A,B) + dist(B,C)
The distances between Descriptors are all bounded by 1, which means they are never greater than 1.
